This is the dominant color of the design. It forms the foundation of the color scheme and covers the majority of the design (60%). The primary color is typically a neutral color and is often used as the background of the design. Colors like white, blue, beige, and so on can be used as primary colors.

This color supports the dominant color and adds more visual interest to the design. It covers about 30% of the design. The secondary color can be used for typography, icons, and subheadings/subtitles in a design. Colors like teal, black, dark blue, and others can be used as secondary colors.

This color usually covers about 10% of the design. It helps highlights specific sections of the design like buttons, call-to-action elements, or any element that needs emphasis in a design. Accent colors are usually colors that have high contrast. Colors like yellow, orange, light green, and others make good accent colors.
